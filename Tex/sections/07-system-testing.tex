\section{System Testing}
This chapter summarizes the testing strategy used for feature validation.

\subsection{Unit Testing}
Unit tests are used for core service functions such as merge logic, page organization logic, coordinate calculations for crop operations, and utility helpers.

\subsection{Integration Testing}
Integration tests validate tool modules and route-level behavior to confirm that user workflows map correctly to processing services.

\subsection{System Testing}
System-level checks include end-to-end behavior of key workflows: file import, operation execution, and output generation. Additional checks include linting, type validation, and build verification.
