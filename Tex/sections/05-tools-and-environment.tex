\section{Tools and Environment Used}
This chapter lists platform requirements and implementation tools.

\subsection{Hardware and Software Requirement}
\subsubsection{Software Requirement}
\begin{itemize}
  \item Operating system: Windows, Linux, or macOS with modern browser support
  \item Runtime: Node.js and pnpm for development
  \item Browser: Chromium-based browser, Firefox, or Safari (latest stable)
  \item Source control: Git
\end{itemize}

\subsubsection{Hardware Requirement}
\begin{itemize}
  \item Processor: dual-core or better
  \item Memory: minimum 4 GB (8 GB recommended)
  \item Storage: minimum 500 MB free space for project and dependencies
\end{itemize}

\subsubsection{Technology to be Used}
\begin{itemize}
  \item React Router, Vite, TypeScript
  \item Tailwind CSS and reusable UI components
  \item pdf-lib and pdfjs-dist for client-side PDF operations
  \item Vitest for automated testing
\end{itemize}
